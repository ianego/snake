How to cross the river 

El problema se basa en un farmero que tiene un problema para poder cruzar un rio teniendo 3 objetos; un costal de maíz, un zorro y un pato. El dilema pasa porque el farmero solo puede cruzar con solo una cosa y el zorro no puede estar junto al pato y el pato no puede estar junto al maíz. 

Resultado:
1. Lleva pato
2. Regresa solo
3. Lleva maíz 
4. Regresa pato 
5. Lleva zorro 
6. Regresa solo 
7. Lleva pato 

En el código lo que se busca hacer es hacer varias pruebas y cada que se llega a un resultado erroneo, este se reinicia hasta encontrar el resultado que es correcto. El programa se corre un n numero de veces hasta que se encuentra el resultado correcto.
